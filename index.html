Política de Privacidade - DarkVoice

1. Introdução

A DarkVoice valoriza a privacidade e a segurança dos seus usuários. Esta política de privacidade explica como tratamos os dados coletados pela extensão DarkVoice, que opera localmente no seu navegador e não compartilha informações com terceiros.

2. Dados Coletados

A DarkVoice não coleta dados pessoais, exceto pela chave de login necessária para autenticação no serviço. Essa chave é armazenada localmente no seu dispositivo e não é compartilhada com servidores externos.

A extensão foi projetada para funcionar completamente no seu navegador, sem enviar dados para a internet. Todos os processamentos são feitos localmente.

3. Uso dos Dados

A única informação coletada, a chave de login, é usada exclusivamente para autenticar o usuário no serviço DarkVoice. Nenhum outro uso é feito desses dados.

4. Segurança

A DarkVoice utiliza práticas de segurança para proteger a chave de login armazenada localmente. No entanto, é importante que você mantenha seu dispositivo seguro para evitar acesso não autorizado.

5. Alterações na Política de Privacidade

Esta política de privacidade pode ser atualizada periodicamente. Recomendamos que você revise esta página regularmente para se manter informado sobre como estamos protegendo suas informações.

6. Contato

Se você tiver dúvidas sobre esta política de privacidade, entre em contato conosco através do e-mail: suporte@darkvoice.com.
